Manual de instalación

1. Descargar Wordrpess

El primer paso para crear nuestra página web es descargar el material necesario, para ello nos dirigiremos a la paǵina oficial de wordpress y descargaremos la versión más reciente.

2. Crear la base de datos

Para poder almacenar toda la información de nuestra página web necesitamos crear una base de datos, con un nombre caracteristico y facil de recordar. Para ello iniciaremos sesión en nuestro gestor de base de datos phpMyAdmin.


Una vez iniciada la sesión en nuestro gestor de base de datos, presionaremos en el botón de "new" e intrduciremos el nombre de la base de datos.


3. Instalar wordrpess

Cuando ya hayamos creado nuestra base de datos procederemos a instalar wordpress. Cuando ingresemos al menú de instalación nos solicitará el nombre de la base de datos, así como el usuario y contraseaña de acceso a nuestra base de datos. También nos solicitará crear un usuario de administrador y un correo.



4. Instalar tema a wordpress

Para subir el tema en la tienda se necesita agregar por ssh pdirigiendonos a la carpeta de wp-content y dirigirse a la carpeta themes, en esta clonaremos el repositorio donde tengamos nuestro tema.

6. Plan de operación de negocio

a) Procesos de producción
Al emepzar el negocio es necesario de realizar una gran inversión para la compra de las ipiezas que se van a vender. b) Productos

Los productos serán registrados en la tienda, ahí estarán guardados en la Base de datos, se configurará la categoría de cada uno de los productos y dandole un precio minimo del 20% añadido del precio de compra del proveedor para generar una ganancia e ir exapndiendo el catalogo.

La adquisición de los productos en venta será obtenidos a partir de los proveedores fabricantes de estas piezas, cada cierto periodo de tiempo se realizará un inventario para conocer la cantidad de producto faltante. Es necesario conocer las ganancias que vayamos obteniendo de lo vendido, debido a que se debe controlar la economía y saber cuando requerir de una nueva inversión de compra de más producto.
